# 研究概述

课题组从事**计算岩土力学**研究，聚焦颗粒材料的多尺度力学行为，发展"微观机理—宏观表征—工程应用"的全链条研究体系。围绕异形颗粒离散元方法、多尺度流固耦合、随机场建模与可靠度分析等核心方向，自主研发 NetDEM 通用颗粒计算软件，服务于海洋资源开发与复杂系统多尺度建模及不确定性量化。

## 研究方向

- [**颗粒材料智能表征与数据驱动建模**](machine-learning) — 深度学习图像重建、虚拟颗粒生成、数据驱动本构
- [**异形颗粒离散元方法与智能计算**](algorithms) — SDF统一接触理论、全流程数字化技术、机器学习加速
- [**多尺度流固耦合与多相流模拟**](multiphase) — 混合解析CFD-DEM框架、相变-破碎耦合、多相流失稳机理
- [**随机场建模与岩土工程可靠度**](random-field) — 智能勘察优化、旋转各向异性、能量演化失稳判据

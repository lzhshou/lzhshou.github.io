# 实验室成员

## 课题组负责人 (Principal Investigator)

### 赖正首 副教授

赖正首，中山大学土木工程学院副教授，2018年获美国克莱姆森大学土木工程博士学位，2012年获中山大学交通工程学士学位。曾在香港科技大学土木与环境工程学院从事博士后研究（"香江学者"计划），并在美国爱达荷国家实验室担任研究实习生。主要研究方向为计算岩土力学，包括有限元、离散元、物质点、随机场、计算流体动力学及多尺度等计算方法，以及先进数值算法与软件研发，应用于过程仿真分析及基础设施防灾减灾研究。主持多项国家级和省部级科研项目，包括国家自然科学基金青年项目和博士后科学基金面上项目，在颗粒介质多尺度模拟和不规则颗粒离散元方法等领域取得了系列研究成果。

电子邮箱：laizhengsh@mail.sysu.edu.cn

# APAAM 实验室
# 应用过程分析与模拟实验室 (Applied Process Analysis And Modeling Laboratory)

欢迎访问应用过程分析与模拟实验室。我们致力于在计算力学和材料科学领域开发创新解决方案。

## 研究方向

- **计算力学**：有限元、离散元、物质点法
- **颗粒材料建模**：基于CT图像的颗粒重建、形状表征、接触检测
- **机器学习应用**：基于深度学习的材料参数识别、结构健康监测
- **高性能计算**：GPU加速、并行计算
- **基础设施健康监测**：基于计算机视觉的结构检测与评估

SAM 国产 CAE 软件应用竞赛

竞赛简介​

SAM 国产结构分析软件应用竞赛（又称"奥蓝托杯"国产工业 CAE 软件创新应用大赛）旨在推进国产工业软件自主可控，验证 SAM 软件在复杂工程问题中的计算精度和适用性。

  • 主办方：中船奥蓝托（北京）科技有限公司
  • 参赛对象：全国高校、科研院所及企业单位
  • 竞赛形式：线上提交作品，组委会评审

2026 第一届：海上风机结构静力与模态分析对比​

研究内容​

作品《基于 SAM 与 Abaqus 的海上风机结构静力与模态分析对比研究》以海上风机为对象，建立包含塔筒、机舱、轮毂及简化叶片的有限元模型，采用 SAM 与 Abaqus 进行静力与模态分析对比。

海上风机整体几何模型

结果表明 SAM 与 Abaqus 在9种典型工况下整体一致性较高：

  • 静力分析：塔顶位移相对误差 2.1%~3.5%，最大等效应力相对误差 1.8%~3.2%
  • 模态分析：各阶固有频率相对误差均小于 1.5%
典型工况应力与位移云图对比

研究验证了国产 CAE 软件 SAM 在风电结构分析中的工程适用性。

竞赛背景​

  • 竞赛名称：第一届SAM国产结构分析软件应用竞赛（又称"奥蓝托杯"国产工业CAE软件创新应用大赛）
  • 时间：2026年3月
  • 参赛规模：103支队伍报名参赛
  • 获奖设置：一等奖1项、二等奖2项、三等奖3项、优秀奖14项

  • 团队：玖探队
  • 队员：卢昱州、刘明靖、陈建营
  • 指导教师：赖正首
  • 作品：《基于 SAM 与 Abaqus 的海上风机结构静力与模态分析对比研究》

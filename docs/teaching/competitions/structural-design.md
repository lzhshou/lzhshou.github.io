# 结构设计竞赛

## 竞赛简介

土木工程学院结构设计竞赛是中山大学土木工程学院主办的院内学科竞赛，旨在培养学生的结构设计与创新能力，加深对结构力学原理的理解。

- **主办方**：中山大学土木工程学院
- **参赛对象**：土木工程学院本科生
- **竞赛形式**：结构设计、制作与加载测试

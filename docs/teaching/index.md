## 教学理念

### 培养目标

我们致力于培养具有以下能力的土木工程专业人才：

1. **扎实的理论基础**
   - 深厚的数学和力学功底
   - 系统的专业知识体系
   - 跨学科的知识整合能力

2. **卓越的实践能力**
   - 编程和数值计算技能
   - 软件工具应用能力
   - 实验和现场操作技能

3. **创新研究思维**
   - 批判性思维能力
   - 解决复杂问题的能力
   - 独立科研工作能力

4. **国际视野**
   - 跟踪国际前沿动态
   - 国际交流与合作能力
   - 学术英语表达能力

## 学生指导

### 本科生指导

**毕业设计**：
- 每年招收3-5名本科生
- 研究方向：离散元方法、机器学习应用、数值模拟
- 要求：具备编程基础（Python/C++）

**科研训练**：
- 大学生创新创业项目
- 学科竞赛指导（力学竞赛、数学建模等）
- 实验室开放项目

### 研究生指导

**博士生培养**：
- 学制：4年
- 培养目标：独立科研能力和高水平论文发表
- 支持：国际会议参与、联合培养机会

**硕士生培养**：
- 学制：3年
- 培养目标：专业技能和工程应用能力
- 支持：项目参与、实习机会

# 教学

## 教授课程

### 本科生课程

#### 海洋工程勘察
本课程介绍海洋工程勘察的基本原理和方法，包括海底地形测量、海洋地质勘察、海洋环境调查等内容，培养学生在海洋工程领域的勘察能力。

**课程内容**：
- 海洋工程勘察概论
- 海底地形测量技术
- 海洋地质勘察方法
- 海洋环境调查与评估
- 勘察数据处理与分析

#### 工程制图与CAD
本课程教授工程制图的基本原理和方法，以及CAD软件的应用，培养学生的空间想象能力和工程图纸绘制能力。

**课程内容**：
- 工程制图基础
- 投影理论与方法
- CAD软件操作
- 三维建模技术
- 工程图纸标准与规范

#### 线性代数
本课程介绍线性代数的基本概念和理论，包括向量空间、线性变换、特征值和特征向量等，为学生后续学习工程数学和计算力学奠定基础。

**课程内容**：
- 矩阵理论
- 向量空间与线性变换
- 特征值与特征向量
- 线性方程组求解
- 工程应用实例

### 研究生课程

#### 高等计算力学
本课程深入探讨计算力学的高级理论和方法，包括有限元方法、边界元方法、无网格方法等，培养学生解决复杂力学问题的能力。

**课程内容**：
- 连续介质力学基础
- 有限元方法理论与实现
- 非线性问题求解
- 多物理场耦合分析
- 高性能计算技术

#### 离散介质计算方法
本课程介绍离散介质力学的基本理论和计算方法，包括离散元法、分子动力学等，培养学生在颗粒材料、岩土工程等领域的研究能力。

**课程内容**：
- 离散介质力学基础
- 离散元方法原理
- 接触力学与碰撞检测
- 颗粒材料建模
- GPU并行计算实现

## 教学理念

我们致力于培养学生的：
- **理论基础**：扎实的数学和力学理论基础
- **实践能力**：编程和数值计算技能
- **创新思维**：解决复杂工程问题的能力
- **国际视野**：跟踪国际前沿研究动态

# 岛礁珊瑚砂在水流中的拖曳力模型研究与数值模拟分析

## 项目概况

- **项目级别**：校级/省级
- **项目成员**：张皓钦、刘明靖
- **指导教师**：赖正首、黄林冲
- **项目周期**：2024.01 - 2024.12

## 研究背景与意义

岛礁珊瑚砂是南海岛礁建设的重要建筑材料，其在水流作用下的水动力特性直接影响岛礁工程的稳定性。拖曳力作为颗粒-流体相互作用的核心参数，准确预测珊瑚砂颗粒的拖曳力对于理解其在波浪、水流作用下的迁移规律具有重要意义。

珊瑚砂颗粒具有形态不规则、内部孔隙发育等特点，传统的球形颗粒拖曳力公式难以准确描述其水动力行为。本项目采用计算流体力学-离散元耦合方法（CFD-DEM），结合球谐函数虚拟颗粒生成技术，建立考虑颗粒形态特征的拖曳力预测模型。

## 研究内容

**规则颗粒沉降验证**
- 建立椭球体、圆柱体等规则颗粒的CFD-DEM沉降模型
- 验证数值方法在标准工况下的准确性
- 分析长宽比、扁平度等形态参数对拖曳力的影响规律

**真实珊瑚砂颗粒数值模拟**
- 基于球谐系数重构真实形态珊瑚砂颗粒
- 进行单颗粒沉降数值试验，获取速度-拖曳力关系
- 分析颗粒表面粗糙度、孔隙分布对水动力响应的影响

**形态-拖曳力相关性分析**
- 提取颗粒几何形态特征参数（球度、凸度、长宽比等）
- 建立形态参数与拖曳力系数的统计关系
- 开发适用于珊瑚砂的拖曳力预测公式

## 技术路线

- **数值方法**：CFD-DEM耦合（OpenFOAM + 自主DEM程序）
- **颗粒建模**：球谐函数重构真实颗粒形态
- **后处理**：Python/MATLAB数据分析
- **验证手段**：室内沉降试验对比

## 创新点

1. **真实形态颗粒CFD-DEM模拟**：突破传统球形颗粒假设，实现不规则形态珊瑚砂的高保真数值模拟
2. **球谐函数颗粒生成**：基于随机场理论生成统计意义上的虚拟珊瑚砂颗粒
3. **形态-力学关联**：建立可量化的形态参数与拖曳力之间的关系模型

## 预期成果

- 建立珊瑚砂颗粒CFD-DEM沉降分析流程
- 形成形态参数与拖曳力的关联模型
- 发表相关学术论文或会议报告

## 项目成果

- 完成规则颗粒与真实珊瑚砂颗粒的数值模拟
- 验证CFD-DEM方法在珊瑚砂水动力分析中的适用性
- 为岛礁工程珊瑚砂迁移预测提供理论基础

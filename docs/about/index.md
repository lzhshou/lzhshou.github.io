# 关于

## 课题组简介

应用过程分析与模拟实验室（APAAM）位于中山大学，致力于计算岩土力学与多尺度数值模拟研究。研究跨越计算力学、岩土工程和材料科学等多个学科领域。

**研究方向**：
- 有限元、离散元、物质点等计算方法
- 随机场、计算流体动力学
- 多尺度模拟方法
- 先进数值算法与软件研发

应用于复杂多物理过程的数值建模、不确定性量化及数据驱动计算研究。


## 招生与合作

课题组长期招收岩土工程、工程力学方向的硕士研究生，也欢迎博士生加入或开展联合培养。研究方向涵盖有限元/离散元/物质点等数值方法、多尺度模拟、机器学习在岩土工程中的应用等。

- 对计算力学、编程或数值模拟有兴趣的同学优先
- 本科生也可联系参与科研训练或毕业设计
- 欢迎国内外同行开展学术合作与交流

如有意向，请发送邮件至 **laizhengsh@mail.sysu.edu.cn**，简要介绍你的背景和兴趣方向。

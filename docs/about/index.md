# 关于我们

## 实验室概况

应用过程分析与模拟实验室（APAAM）位于中山大学，致力于开发创新的计算方法和软件工具，用于解决复杂工程问题。我们的研究跨越多个学科，包括计算力学、岩土工程和材料科学。

我们的主要研究方向包括：
- 计算力学（有限元、离散元、物质点法）
- 颗粒材料建模
- 工程中的机器学习应用
- 高性能计算
- 基础设施健康监测

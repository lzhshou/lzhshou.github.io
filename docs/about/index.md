# 关于

## 课题组简介

**应用过程分析与模拟实验室（APAAM）**位于中山大学，致力于计算岩土力学与多尺度数值模拟研究。研究跨越计算力学、岩土工程和材料科学等多个学科领域。

**研究方向**：
- 有限元、离散元、物质点等计算方法
- 随机场、计算流体动力学
- 多尺度模拟方法
- 先进数值算法与软件研发

应用于应用过程仿真分析及基础设施防灾减灾研究。

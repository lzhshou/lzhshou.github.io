# 关于我们

## 实验室简介

**应用过程分析与模拟实验室（APAAM Lab）**位于中山大学圿市市校区，致力于计算岩土力学与多尺度数值模拟研究。我们的研究跨越计算力学、岩土工程和材料科学等多个学科领域。

### 核心价值观

- **创新驱动**：开发原创性的数值方法和软件工具
- **实践导向**：面向重大工程问题和国家需求
- **开放合作**：与国内外高校和研究机构开展深度合作
- **人才培养**：培养具有国际视野的创新型人才

# 资源

## 软件工具

### 计算力学软件
- **YADE**: 开源离散元方法软件包，适用于颗粒材料建模
- **LIGGGHTS**: 基于LAMMPS的离散元方法软件
- **FEniCS**: 开源有限元计算平台
- **OpenFOAM**: 开源计算流体动力学软件

### 机器学习工具
- **TensorFlow**: 深度学习框架
- **PyTorch**: 机器学习库
- **scikit-learn**: Python机器学习库
- **OpenCV**: 计算机视觉库

### 数值计算工具
- **MATLAB**: 数值计算和可视化平台
- **Python**: 科学计算生态系统（NumPy, SciPy, Matplotlib）
- **Julia**: 高性能科学计算语言
- **Paraview**: 科学数据可视化软件

## 数据集

### 颗粒材料数据
- **CT扫描数据**: 各种颗粒材料的CT图像数据
- **形状数据库**: 不规则颗粒形状参数数据库
- **接触力数据**: 离散元模拟的接触力数据

### 基准问题
- **标准测试案例**: 验证数值方法的标准问题
- **对比数据**: 实验与数值模拟对比数据

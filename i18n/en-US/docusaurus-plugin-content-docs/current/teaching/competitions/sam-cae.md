# SAM CAE Software Competition

## Overview

The SAM Structural Analysis Software Application Competition (also known as the "Aulento Cup" Domestic Industrial CAE Software Innovation Application Competition) aims to promote the independent control of domestic industrial software and verify the computational accuracy and applicability of SAM software in complex engineering problems.

- **Organizer**: CSSC Aulento (Beijing) Technology Co., Ltd.
- **Participants**: Universities, research institutes, and enterprises nationwide
- **Format**: Online submission, committee review

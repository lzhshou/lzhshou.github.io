## Course Description

Linear Algebra is an essential mathematical foundation course for science and engineering students. It provides mathematical tools for subsequent engineering computation and scientific research, including matrix theory, vector spaces, and linear transformations.

## Learning Objectives

1. Master basic operations of matrices and determinants
2. Understand concepts of vector spaces and linear transformations
3. Master methods for solving systems of linear equations
4. Understand applications of eigenvalues and eigenvectors

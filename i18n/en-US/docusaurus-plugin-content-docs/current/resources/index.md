
# Resources

## Software Tools

### Computational Mechanics Software
- **YADE**: Open-source discrete element method software package for granular material modeling
- **LIGGGHTS**: Discrete element method software based on LAMMPS
- **FEniCS**: Open-source finite element computing platform
- **OpenFOAM**: Open-source computational fluid dynamics software

### Machine Learning Tools
- **TensorFlow**: Deep learning framework
- **PyTorch**: Machine learning library
- **scikit-learn**: Python machine learning library
- **OpenCV**: Computer vision library

### Numerical Computing Tools
- **MATLAB**: Numerical computing and visualization platform
- **Python**: Scientific computing ecosystem (NumPy, SciPy, Matplotlib)
- **Julia**: High-performance scientific computing language
- **Paraview**: Scientific data visualization software

## Datasets

### Granular Material Data
- **CT Scan Data**: CT image data of various granular materials
- **Shape Database**: Irregular particle shape parameter database
- **Contact Force Data**: Contact force data from discrete element simulations

### Benchmark Problems
- **Standard Test Cases**: Standard problems for validating numerical methods
- **Comparison Data**: Experimental and numerical simulation comparison data

# Research Areas

## Computational Mechanics

We focus on developing and applying advanced numerical methods to solve complex engineering problems:

- **Finite Element Method (FEM)**: For continuum mechanics analysis
- **Discrete Element Method (DEM)**: For granular materials and discontinuous media analysis
- **Material Point Method (MPM)**: For large deformation and multiphase flow problems
- **Multiscale Modeling**: Bridging microscopic and macroscopic scales

## Granular Materials Modeling

Based on advanced image processing and numerical modeling techniques:

- **CT Image Reconstruction**: Reconstructing real particle shapes from medical or industrial CT images
- **Shape Characterization**: Developing new particle shape descriptors and quantification methods
- **Contact Detection Algorithms**: Efficient contact detection algorithms for irregular particles
- **Multiscale Modeling**: Cross-scale modeling from particle scale to macroscopic material properties

## Machine Learning Applications

Applying artificial intelligence technologies to civil engineering:

- **Material Parameter Identification**: Deep learning-based material constitutive parameter inversion
- **Structural Health Monitoring**: Computer vision-based structural damage detection
- **Predictive Modeling**: Machine learning-based engineering performance prediction
- **Optimization Design**: Intelligent optimization algorithms in engineering design

## High-Performance Computing

Developing efficient parallel computing algorithms and software:

- **GPU-Accelerated Computing**: Leveraging GPU parallel computing power to accelerate numerical simulations
- **Distributed Computing**: Large-scale parallel computing framework development
- **Algorithm Optimization**: Performance optimization and acceleration of numerical algorithms
- **Software Development**: Development and maintenance of open-source scientific computing software

## Infrastructure Health Monitoring

Based on modern sensing technologies and data analysis methods:

- **Computer Vision Detection**: Image and video-based structural inspection technologies
- **Sensor Networks**: Multi-sensor fusion monitoring systems
- **Data Analysis**: Big data analysis applications in infrastructure monitoring
- **Warning Systems**: Intelligent warning and decision support systems

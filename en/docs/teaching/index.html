教学与指导

招生信息​

招收对象​

博士研究生：

  • 岩土工程方向
  • 工程力学方向

硕士研究生：

  • 土木水利（专业学位）
  • 工程力学（学术学位）

本科科研助理：

  • 大二及以上本科生
  • 对科研有浓厚兴趣

申请要求​

  1. 专业基础

    • 土木工程、工程力学、机械工程等相关专业
    • 扎实的数学和力学基础

  2. 编程能力（满足其一）

    • 熟悉 Python
    • 熟悉 C/C++
    • 熟悉 MATLAB

  3. 英语能力

    • 能够阅读英文文献
    • 能够撰写英文学术论文

  4. 其他要求

    • 对计算力学有浓厚兴趣
    • 具备良好的学习能力和科研素养

研究兴趣匹配​

我们欢迎具有以下研究兴趣的学生：

  • 计算力学：有限元、离散元、物质点法等数值方法
  • 多尺度模拟：从微观到宏观的跨尺度建模
  • 机器学习：AI在土木工程中的应用
  • 软件研发：科学计算软件开发

培养目标​

  • 理论基础：扎实的数学和力学理论功底
  • 实践能力：编程实现和数值计算技能
  • 创新思维：独立解决科研问题的能力
  • 国际视野：跟踪国际前沿、参与国际交流
